Welcome to the CleanerCSV datafile cleaner

The app will load a specified CSV file and perform various cleaning tasks as specified in the selected JSON specifications file. Profile reports of the source and the output data are created for comparison.

Start

by clicking the "Load specs" button above to select a specification file. Once the spec and source files have been successfully loaded clic the "Clean data" button to run the task specified in the spec file.

Please be patient

Note, that some operations, profiling in particular but also some column conversions might take a significant amount of time on a large datasample.

Command line launch

The app can be launched from the command line using

python.exe app <specs_filename.json>

The program will proceed to load the specified data, perform cleaning opeating and save the data as specified in the specs file before terminating.

Spec files

The specs are stored in JSON format.
The app folder always has an example default_specs.json file. This file will be rewritten if deleted and loaded is no valid specs file is provided.

Spec file options

Note - All options except input_file, output_file, delimiter_in, delmiter_out are optional and assumed false or empty if not specified.
input_file, output_file : path/fileanme.csv - The source and target CSV data files.
delimiter_in, delmiter_out : character - Assumed "," if omitted. - Data separators for source and target scv files respectively.
input_file_rofile, output_file_profile : path/filename.html - HTML Filenames for profiles for sourc eand output data. summary_file : path/filename - Filename to save a copy of the cleaning log, for example keeping it with the data or profile files.
Note - regardles of this option an addtional full logfile will alway be written as data_cleaner.log within the application folder.
drop_repeat_headers : true|false - Will remove possible dupplicates of the header line.
drom_dupplicates : true|false - Will remove dupplicate data rows from the dataframe.
drop_na : true|false - Will remove data rows containing NaN and NaT (not a number and not a time) from the data.
clean_types : true|false - Will convert data rows to numerical types or specified formats.
Note - This option requires some or all of the below options to specify columns and data types for conversion.
str_col, float_col, int_col, numeric_col : [list of column names] - column labels to be converted to the specified types.
Note - Explicit numeric conversions might fail, if the data is unsuitable. Running the generic numeric conversion with drop_na : true mitigates this risk.
datetime_col, datetime_format [lists] - Columns to be converted as date and time. Requires a datetime format string (specifying the input data format)for each column. drop_col - Columns to be dropped form the dataframe.
